# Noesis — Visuals

Standalone, reusable diagrams for the pitch deck, plain-language paper, and slides.
Each is a self-contained SVG: open in any browser, drop into a deck, or place in a doc. Legible at slide and paper size.

**Consistent visual language**
- **Palette** — ink `#1a1f2e` (text), slate `#5b6478` (secondary), hairline `#d9dde6`. War / zero-sum = muted red `#c2453d`. Noesis / accretion = teal `#1f9d8f`. The conserved prize = gold `#e0a83e`. Paper `#fbfaf7`.
- **Type** — system sans, one weight scale (700 headers, 600 labels, regular body).
- **Convention** — red = the old zero-sum fight; teal = Noesis conserving/absorbing; arrows colliding = waste, arrows flowing in = accretion. In the status diagram: **solid = built**, **dashed = designed**, **faded = open / null-tested**.

| File | Caption |
|---|---|
| `01-zero-sum-vs-accretion.svg` | Fight over a fixed pie (arrows colliding) vs add to a growing graph (arrows flowing in, conserved). |
| `02-reverse-fork.svg` | Splitting off (a fork diverges, value splits) vs merging in (a reverse-fork converges, value conserved). |
| `03-claimable-attribution-funnel.svg` | The whole existing contribution graph -> prove it's you (the filter) -> wallet created + credit claimed -> a real builder onboarded. |
| `04-two-level-geometry.svg` | The same move at two zoom levels: whole chains converge into one graph, and individual people/repos converge into the same graph. |
| `05-prize-swap.svg` | Scarce prize (everyone grabbing one coin, someone loses) -> conserved prize (every piece kept and stacked); change the prize, dissolve the war. |
| `06-credit-precedes-adoption.svg` | Other chains: join -> then earn from zero. Noesis: already credited -> join to claim. Credit comes first and pulls adoption behind it. |
| `07-honesty-status.svg` | What's real, drawn, open: conservation core BUILT & TESTED, convergence adapter DESIGNED NOT YET BUILT, learned value measure FIRST REAL-DATA TEST NULL (unsupported, not refuted). |
| `08-war-that-ends-itself.svg` | A battlefield where every chain attacks every other; Noesis sits in the center with no outgoing attacks, only incoming merge-arrows. |

**Honesty note (non-negotiable):** `07-honesty-status.svg` carries the STATUS-LEDGER wording verbatim — conservation core = *demonstrated*, convergence adapter = *designed/unbuilt*, learned value measure = *null-tested*. No diagram here implies live deployment, joined chains, or a proven value-measure moat.
